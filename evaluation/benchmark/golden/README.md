# Golden Set

Curated, expert-graded questions with annotated expected-evidence IDs
(`../annotations/`). Unlike the holdout, the golden set is used during development to
guide tuning.

Do not commit restricted source content. Store pointers (document IDs + coordinates) and
expected-evidence references, not the documents themselves.

Building this depends on the fixed equipment family, workflow, and canonical identity
source — see the open decisions in the plan. This is the critical-path dependency for the
whole evaluation gate.
